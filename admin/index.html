Sadece site sahibi. Token tarayıcınızda saklanır, sunucuya gönderilmez.

Token oluşturmak için: GitHub → Settings → Developer settings → Fine-grained tokens → Generate: Only select repositories altından site deposunu seçin, Repository permissions → Contents: Read and write verin. (Alternatif: classic token, repo yetkisi.)
